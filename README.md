# skilvul-be

## Endpoint:

Base URL: https://skilvul-be-production.up.railway.app/api/

- `GET` - `/todos` - Get all todos.
- `POST` - `/todos` - Add new todo.
- `PUT` - `/todos/{id}` - Edit todo.
- `DELETE` - `/todos/{id}` - Delete todo.
- `GET` - `/todos/{id}` - Get single todo.

## Feature

- Get all todos.
- Add new todo.
- Edit todo.
- Delete todo.
- Get single todo.

## Stack

- Backend: ExpressJS.
- Database: MongoDB.
- ODM: Mongoose.
- Deployment: Railway.
- Architecture: MVC (OOP).
